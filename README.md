# JMonkeyEngine Mobile Starter

🚀 Create 3D games directly on your mobile device.

Base project configured for developing 3D games using JMonkeyEngine and Java, fully on a smartphone — no PC required.

## 📱 About
This repository provides a functional starter structure for creating 3D games directly on mobile devices, without the need for a computer.

## 🚀 Technologies
- Java
- JMonkeyEngine 3.3.2
- Android
- Code On The Go / Code Assist / AIDE

## 🎮 Purpose
To simplify the development and learning process of mobile 3D game development in a practical, fast, and accessible way.

## 📂 Structure
- Assets (models, textures)
- Base game code
- Ready-to-run setup (sky and ground)

## 💡 Ideal for
- Beginner game developers
- Mobile development
- Rapid prototyping
- People without access to a PC setup
- Computer graphics studies

## ▶️ How to use
1. Clone the repository  
2. Open it in your Android environment  
3. Run the project  
4. Start coding  

## 📌 Notes
Simple and lightweight project focused on learning and experimentation.
